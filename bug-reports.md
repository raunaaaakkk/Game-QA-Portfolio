# Bug Report 1

## Bug ID
BUG-001

## Title
Player Character Stuck Between Collision Objects

## Environment
Windows 11  
Keyboard & Mouse  
1080p Resolution

## Steps to Reproduce
1. Launch the game.
2. Move the player character toward the crate near the spawn point.
3. Jump diagonally into the corner between the crate and the wall.

## Expected Result
Player should slide away from the object or land normally without obstruction.

## Actual Result
Player becomes stuck between the crate and wall and cannot move.

## Severity
Medium

## Frequency
Occurs approximately 3 out of 5 attempts.

# Bug Report 2

## Bug ID
BUG-002

## Title
Player Can Partially Pass Through Environment Boundary

## Environment
Windows 10 / Windows 11  
Keyboard Input

## Steps to Reproduce
1. Start the game.
2. Move the player character toward the edge of the map.
3. Continue holding forward movement against the boundary.

## Expected Result
Player should stop at the map boundary.

## Actual Result
Player partially clips through the boundary wall before being pushed back.

## Severity
Low

## Frequency
Occurs occasionally.

# Bug Report 4

## Bug ID
BUG-004

## Title
Camera Jitter When Player Collides With Objects

## Environment
Windows 11  
Keyboard & Mouse

## Steps to Reproduce
1. Start the game.
2. Move player directly into a solid object such as a crate or wall.
3. Rotate the camera while pushing against the object.

## Expected Result
Camera should remain stable and follow player smoothly.

## Actual Result
Camera briefly jitters and shifts unexpectedly.

## Severity
Low

## Frequency
Occurs intermittently.

# Bug Report 5

## Bug ID
BUG-005

## Title
Player Movement Continues Briefly After Key Release

## Environment
Windows 10 / Windows 11

## Steps to Reproduce
1. Launch the game.
2. Move player forward.
3. Release movement key quickly.

Low
